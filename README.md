# House Blog! #

Find it at [https://house.youngram.com].

Runs on Jekyll and hacks.

## Compose commands ##

    jekyll draft NAME
    jekyll post NAME
    jekyll publish NAME
    jekyll unpublish NAME

## Local preview ##

    jekyll serve -D

## Installing rmagick on Arch ##

    pacman -S libmagick6
    export PKG_CONFIG_PATH="/usr/lib/imagemagick6/pkgconfig"
    bundle
